# Cocovs Tap

## How do I install these formulae?

`brew install cocovs/tap/<formula>`

Or `brew tap cocovs/tap` and then `brew install <formula>`.

For this project:

```bash
brew install cocovs/tap/batchjob-cli
```

`batchjob-cli` installs the CLI only. If you also want the Codex or Claude skill pack, use the GitHub Release installer from `batchjob-agent-kit`.
